# logica-de-programacao
Exercícios do curso de lógica de algoritmo organizados por capítulos.
Curso de Algoritmos e Lógica de Programação (Udemy)

Este é um curso completo e didático que ensina os fundamentos essenciais de lógica de programação e algoritmos, começando pelo VisualG e chegando até conceitos que permitem programar em outras linguagens com mais facilidade.

O que o curso cobre:

Conceitos básicos de algoritmos e lógica de programação.
Uso de variáveis, entrada e saída de dados, operadores e expressões.
Estruturas de controle como sequência, decisão (condicionais) e repetição.
Trabalhar com vetores e matrizes.
Testes de mesa e depuração para entender e corrigir o fluxo do programa.
Aplicação de lógica em diferentes linguagens como C, C++, Python, C# e Java.

- Os exercícios seguem o curso da Udemy e servem **apenas para prática e fixação**.
- Não são projetos finais, mas representam a **consolidação da base de lógica de programação**.
